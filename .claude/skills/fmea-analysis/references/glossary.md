## [REF] 용어 사전

### 철심 (Core) 관련

| 전문용어 | 쉬운표현 | 영문 | 설명 |
|---------|---------|------|------|
| **자왜변형** | 철심진동 | Magnetostriction | 자속에 의해 철심이 미세하게 변형되는 현상 |
| **자왜진동** | 자속진동 | Magnetostrictive vibration | 자왜현상으로 발생하는 진동 |
| **층간단락** | 철심내부합선 | Interlaminar short | 적층된 철심판 사이의 절연 파괴 |
| **와전류** | 유도전류 | Eddy current | 자속 변화로 철심에 유도되는 전류 |
| **철손** | 철심손실 | Core loss | 철심에서 발생하는 에너지 손실 |
| **히스테리시스손** | 자화손실 | Hysteresis loss | 자화 방향 변화 시 발생하는 손실 |
| **적층이완** | 철심판풀림 | Lamination loosening | 적층된 철심판의 결합이 느슨해짐 |
| **적층갭** | 철심판틈새 | Lamination gap | 적층된 철심판 사이의 공극 |
| **T-Joint** | 요크접합부 | T-Joint | 철심 Leg와 Yoke가 만나는 부분 |
| **Step-Lap** | 계단형겹침 | Step-Lap joint | 철심 접합부의 계단형 구조 |
| **자기포화** | 자속과다 | Magnetic saturation | 철심이 더 이상 자속을 수용 못함 |
| **버(Burr)** | 철심날카로운모서리 | Burr | 철심판 절단면의 날카로운 돌출부 |

### 권선 (Winding) 관련

| 전문용어 | 쉬운표현 | 영문 | 설명 |
|---------|---------|------|------|
| **절연열화** | 절연성능저하 | Insulation degradation | 절연재 성능이 저하됨 |
| **절연파괴** | 절연깨짐 | Insulation breakdown | 절연재가 더 이상 절연 기능 못함 |
| **권선변형** | 권선찌그러짐 | Winding deformation | 권선 형상이 변형됨 |
| **단락전류** | 합선전류 | Short-circuit current | 단락 시 흐르는 과대 전류 |
| **부분방전** | 미세방전 | Partial discharge (PD) | 절연체 내 국부적 방전 |
| **턴간단락** | 권선내부합선 | Turn-to-turn short | 같은 권선 내 인접 턴 간 합선 |
| **층간절연** | 층간막 | Layer insulation | 권선 층 사이의 절연 |
| **접촉불량** | 접촉안됨 | Poor contact | 전기적 접촉이 불완전함 |

### 절연유/냉각 관련

| 전문용어 | 쉬운표현 | 영문 | 설명 |
|---------|---------|------|------|
| **DGA** | 유중가스분석 | Dissolved Gas Analysis | 절연유 내 용존 가스 분석 |
| **열화** | 성능저하 | Deterioration | 시간에 따른 성능 저하 |
| **누유** | 기름누출 | Oil leakage | 절연유가 외부로 새어나옴 |
| **슬러지** | 찌꺼기 | Sludge | 절연유 열화로 생기는 침전물 |
| **열점** | 국부과열점 | Hot spot | 국부적으로 온도가 높은 지점 |
| **유로** | 기름흐름길 | Oil duct | 냉각유가 흐르는 통로 |

### 기계적 현상

| 전문용어 | 쉬운표현 | 영문 | 설명 |
|---------|---------|------|------|
| **피로파손** | 반복하중파손 | Fatigue fracture | 반복 하중으로 인한 파손 |
| **크리프** | 서서히변형 | Creep | 지속 하중 하에 서서히 변형 |
| **응력집중** | 힘집중 | Stress concentration | 특정 부위에 응력이 집중됨 |
| **좌굴** | 휨변형 | Buckling | 압축력으로 휘어짐 |
| **토크** | 조임힘 | Torque | 회전력, 조임력 |

### 시험/검사 관련

| 전문용어 | 쉬운표현 | 영문 | 설명 |
|---------|---------|------|------|
| **FAT** | 공장출하시험 | Factory Acceptance Test | 출하 전 공장 시험 |
| **SAT** | 현장인수시험 | Site Acceptance Test | 현장 설치 후 시험 |
| **내전압시험** | 고압견딤시험 | Withstand voltage test | 절연 내력 시험 |
| **유도시험** | 유도내전압시험 | Induced voltage test | 유도 전압에 의한 시험 |
| **SFRA** | 주파수응답분석 | Sweep Frequency Response | 권선 변형 진단 기법 |
| **FRA** | 주파수응답 | Frequency Response Analysis | 주파수별 응답 분석 |
| **임펄스시험** | 뇌서지시험 | Impulse test | 낙뢰 충격 모사 시험 |

### 평가 용어

| 전문용어 | 쉬운표현 | 영문 | 설명 |
|---------|---------|------|------|
| **RPN** | 위험우선순위 | Risk Priority Number | S*O*D 계산값 |
| **AP** | 조치우선순위 | Action Priority | H/M/L 등급 |
| **심각도(S)** | 영향심각성 | Severity | 고장영향의 심각한 정도 |
| **발생도(O)** | 발생빈도 | Occurrence | 고장원인의 발생 빈도 |
| **검출도(D)** | 검출난이도 | Detection | 고장원인 검출 어려움 정도 |

### [!!] 라이프사이클 단계 약어 (FMEA H열/J열 필수!)

> **[CRITICAL]** FMEA 작성 시 아래 약어를 정확히 사용!

| 약어 | 정식 명칭 | 영문 | 설명 | FMEA 활용 |
|------|----------|------|------|----------|
| **DR** | 설계검토 | Design Review | 설계 단계에서 도면/계산서 검토 | H열: 예방대책, J열: 검출대책 |
| **수입검사** | 수입검사 | Incoming Inspection | 재료/부품 입고 시 검사 | J열: 검출대책 |
| **공정검사** | 공정검사 | In-Process Inspection | 제조 중 검사 | J열: 검출대책 |
| **FAT** | 공장출하시험 | Factory Acceptance Test | 출하 전 공장 시험 | J열: 검출대책 |
| **SAT** | 현장인수시험 | Site Acceptance Test | 현장 설치 후 시험 | J열: 검출대책 |

### [!!] 단계 태그 사용법 (H열/J열)

> **[LOCK]** H열/J열 작성 시 단계 태그 필수! 대괄호 금지!

**예방대책(H열) 단계 태그**:
| 태그 | 의미 | 예시 |
|------|------|------|
| `설계:` | 설계 단계 예방 | `설계: 설계검토 (DR)` |
| `재료:` | 재료 관리 예방 | `재료: 경도 규격 관리` |
| `제작:` | 제조 단계 예방 | `제작: 토크 관리` |
| `시험:` | 시험 단계 예방 | `시험: 내전압시험 적용` |

**검출대책(J열) 단계 태그**:
| 태그 | 의미 | 예시 |
|------|------|------|
| `설계:` | 설계검토(DR)로 검출 | `설계: 설계검토 (DR) 승인` |
| `수입검사:` | 수입검사로 검출 | `수입검사: 경도 측정` |
| `공정검사:` | 공정검사로 검출 | `공정검사: 치수 검사` |
| `시험:` | FAT/SAT로 검출 | `시험: 내전압시험 (FAT)` |

### [OK] H열/J열 작성 예시

**[OK] 올바른 예시** (약어 정확히 사용):
```
설계: 설계검토 (DR)
- 압착 사양 적정성 검토
- CHECK SHEET 항목 확인
```

**[X] 잘못된 예시** (약어만 사용, 설명 없음):
```
설계: 설계검토 (DR)    <- "DR"만 쓰면 의미 불명확!
```

**[OK] 권장 형식** (약어 + 설명):
```
설계: 설계검토 (Design Review)
- 압착 사양 적정성 검토 (단자 CHECK SHEET 단자조립-No.1)
- 압착력 계산서 검증 (TD4 p.85)
```

## 📝 병기 예시

### 고장형태 (E열)

| 병기 전 | 병기 후 |
|--------|---------|
| 자왜변형 | 유해: 자왜변형(철심진동) |
| 층간단락 | 부족: 층간단락(철심내부합선) |
| 적층이완 | 부족: 적층이완(철심판풀림) |

### 고장원인 (F열)

| 병기 전 | 병기 후 |
|--------|---------|
| 설계: 자왜특성 미고려 | 설계: 자왜특성(철심진동특성) 미고려 |
| 재료: 와전류 손실 과다 | 재료: 와전류(유도전류) 손실 과다 |

### 고장메커니즘 (G열)

| 병기 전 | 병기 후 |
|--------|---------|
| 자왜변형 → 진동 → 소음 | 자왜변형(철심진동) → 진동 증폭 → 소음 발생 |
| 와전류 → 발열 → 열화 | 와전류(유도전류) → 발열 → 절연열화(성능저하) |
