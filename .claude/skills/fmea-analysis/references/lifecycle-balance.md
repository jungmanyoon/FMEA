## 시험 단계 원인 유형 참고

시험 단계 원인을 찾을 때 참고할 수 있는 5가지 유형입니다.
**WebSearch에서 관련 항목이 발견될 때만 사용하세요.**

---

## 시험 단계 5가지 원인 유형

**1. 시험 누락 (Test Omission)**
```
예시:
- 시험: 절연 저항 시험 누락
- 시험: 무부하 손실 측정 누락
- 시험: 내전압 시험 생략
- 시험: 온도 상승 시험 미실시
- 시험: 부분 방전 시험 누락
```

**2. 시험 방법 오류 (Test Method Error)**
```
예시:
- 시험: 시험 전압 설정 오류
- 시험: 측정 장비 교정 불량
- 시험: 시험 절차 오류
- 시험: 샘플링 방법 부적절
- 시험: 환경 조건 미준수 (온도/습도)
```

**3. 판정 기준 오류 (Acceptance Criteria Error)**
```
예시:
- 시험: 합격 기준 오판정
- 시험: 측정 오차 미고려
- 시험: 판정 기준 적용 오류
- 시험: 허용 한계 설정 오류
- 시험: 데이터 분석 오류
```

**4. 측정 불량 (Measurement Error)**
```
예시:
- 시험: 센서 부착 위치 오류
- 시험: 측정 장비 정밀도 부족
- 시험: 데이터 기록 오류
- 시험: 계측기 영점 오차
- 시험: 측정 시점 부적절
```

**5. 검사 불량 (Inspection Error)**
```
예시:
- 시험: 외관 검사 누락
- 시험: 육안 검사 한계
- 시험: 비파괴 검사 미실시
- 시험: 치수 검사 오류
- 시험: 표면 결함 미검출
```

## 시험 단계 원인 생성 프로세스

**Step 1: 각 고장형태별 시험 가능성 확인**

| 고장형태 | 시험 단계 원인 예시 |
|----------|---------------------|
| **부족** | 시험: 성능 측정 누락, 기준치 미달 미검출 |
| **과도** | 시험: 과전류 시험 누락, 온도 상승 미측정 |
| **저하** | 시험: 경년 열화 예측 시험 미실시, 수명 시험 누락 |
| **간헐** | 시험: 부하 변동 시험 누락, 동특성 시험 미실시 |
| **의도하지않음** | 시험: 오배선 검사 누락, 극성 시험 오류 |
| **부재** | 시험: 구성품 누락 검사 미실시, 조립 확인 누락 |
| **지연** | 시험: 응답 시간 측정 누락, 동작 속도 미검증 |
| **역효과** | 시험: 부작용 확인 시험 누락, 간섭 시험 미실시 |

**Step 2: 각 기능별 필수 시험 항목 생성**

**예시: 규소강판 (GOES)**

```
기능 1: 자속 유도
  └─ 필수 시험:
      - 무부하 손실 측정 (IEC 60404-8-4)
      - 자속 밀도 측정 (IEC 60404-3)
      - 투자율 측정 (IEC 60404-4)
      - 철손 측정 (Doble M4000)
  └─ 시험 단계 원인:
      - 시험: 무부하 손실 측정 누락
      - 시험: 자속 밀도 측정 오류
      - 시험: 측정 주파수 설정 오류
      - 시험: 철손 측정 장비 교정 불량

기능 2: 인장 응력
  └─ 필수 시험:
      - 인장 시험 (IEC 60404-8-6)
      - 항복 강도 시험
      - 연신율 측정
  └─ 시험 단계 원인:
      - 시험: 인장 시험 미실시
      - 시험: 시험편 가공 불량
      - 시험: 하중 측정 오류

기능 3: 자손실 저감 최소화
  └─ 필수 시험:
      - 와전류 손실 측정 (Doble M7000)
      - 히스테리시스 손실 측정
      - 총 철손 측정 (IEC 60404-2)
  └─ 시험 단계 원인:
      - 시험: 와전류 손실 측정 누락
      - 시험: 고조파 손실 측정 미실시
      - 시험: 측정 온도 조건 미준수
```

**Step 3: WebSearch 결과에서 시험 관련 정보 추출**

**IEC/IEEE 표준에서**:
- IEC 60076-1: Factory tests (공장 시험)
- IEC 60076-3: Insulation test procedures (절연 시험)
- IEEE C57.12.90: Test code for transformers (시험 규격)
- IEEE C57.125: Failure investigation (고장 조사 = 시험 누락 발견)

**Doble Engineering에서**:
- 진단 시험 절차 및 장비 (M4000, M5000, M7000)
- 시험 누락 시 발생 가능한 고장 사례
- 측정 오차 및 판정 오류 사례

**Transformer Magazine에서**:
- 시험 누락으로 인한 실제 고장 사례
- 잘못된 시험 방법으로 인한 미검출 사례

## 시험 단계 원인 생성 실전 예시

**고장형태: 과도 (와전류 증가)**

**설계 단계 원인**:
```
설계: 자속 밀도 설계 오류
설계: 적층 두께 설계 부족
설계: 절연 코팅 두께 설계 오류
```

**재료 단계 원인**:
```
재료: 전기 저항 부족
재료: 절연 코팅 재질 불량
재료: 규소 함량 부족
```

**제작 단계 원인**:
```
제작: 층간 절연 손상
제작: 적층 압착력 부족
제작: 절연 코팅 파손
```

**시험 단계 원인** ([OK] 추가 필수!):
```
시험: 와전류 손실 측정 누락 (Doble M7000)
시험: 무부하 손실 시험 미실시
시험: 철심 접지 전류 측정 누락 (IEEE C57.125)
시험: 절연 저항 시험 판정 오류
시험: 온도 상승 시험 조건 미준수
```

## 4단계 분류 참고

**각 단계에서 찾을 수 있는 원인 유형** (WebSearch 검색 키워드 참고용):
- **설계 단계**: 설계 기준 오류, 계산 실수, 표준 미준수
- **재료 단계**: 재질 불량, 자재 불량, 입고 검사 누락
- **제작 단계**: 가공 불량, 조립 불량, 용접 불량, 치수 불량
- **시험 단계**: 시험 누락, 측정 오류, 판정 오류, 검사 불량

**[!] 주의**:
- 비율을 맞추기 위해 항목을 만들지 말 것
- WebSearch에서 발견된 항목만 해당 단계로 분류
- 특정 단계가 0개여도 실제로 해당 원인이 없으면 OK
