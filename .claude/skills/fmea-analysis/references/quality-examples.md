## 고장메커니즘 작성 패턴

### [X] 피해야 할 표현

**문제 1: 인과관계 불명확**
- "와전류 증가" (왜 증가하는지 불명)
- "코팅 불량" (현상만 나열)
- "성능 저하" (어떻게 저하되는지 불명)

**문제 2: 정량화 없음**
- "온도가 상승한다" → 얼마나?
- "손실이 발생한다" → 몇 %?
- "강도가 약해진다" → 어느 정도?

**문제 3: 물리적 과정 생략**
- "버 발생으로 단락" (중간 과정 누락)
- "습기로 부식" (메커니즘 생략)

### [OK] 권장 표현 패턴 (FMEA 실무 표준)

**[!] FMEA 전문가 관점: 간결하고 명확하게!**

**핵심 원칙**:
- **구조**: 원인 → 과정 → 결과 (3단계 논리적 연결)
- **화살표(→)**: 인과관계 표현에 활용
- **WebSearch 기반**: 실제 문헌에서 확인된 메커니즘 사용

**패턴 1: 간단한 인과관계 체인**

형식: `원인 → 진행 과정 → 최종 결과`

**예시 1 - 와전류 손실** (변압기 Core):
```
적층 불량 → 와전류 손실 증가 → 국부 과열
```
(24자, 실무 표준)

**예시 2 - 층간 절연 파괴**:
```
절연 코팅 열화 → 층간 단락 → 와전류 증가
```
(23자, 간결하고 명확)

**패턴 2: 환경 요인 + 물리 변화**

형식: `환경 조건 → 물리 변화 → 기능 저하`

**예시 3 - 부식**:
```
습기 침투 → 철심 부식 → 자기 특성 저하
```
(21자, 이해하기 쉬움)

**예시 4 - 열화**:
```
과부하 운전 → 절연유 열화 → 절연 성능 저하
```
(24자, 명확한 인과관계)

**패턴 3: 구조/재료 결함**

형식: `결함 조건 → 메커니즘 → 고장 발생`

**예시 5 - 피로 파손** (클램프):
```
체결력 부족 → 진동 발생 → 피로 균열
```
(20자, 핵심만 표현)

**예시 6 - 기계적 파손**:
```
철심 재질 불량 → 응력 집중 → 균열 발생
```
(22자, 간단명료)

**[OK] 정량화가 필요한 경우** (선택적):

일부 경우에만 정량화를 추가할 수 있습니다:

**예시 7 - 부식 (정량화 포함)**:
```
습기 침투 → 50μm 부식층 형성 → 자속 감소
```
(26자, 필요시 정량화)

**예시 8 - 열 발생 (정량화 포함)**:
```
와전류 증가 → ΔT=30℃ 온도 상승 → 절연 파괴
```
(27자, 온도 명시)

**[!] 피해야 할 과도한 표현**:
```
[X] 적층 두께 0.23mm→0.5mm 증가 → 와류 전류 I∝(dΦ/dt)×A에서 I 증가 → I²R 손실 4배 증가
```
→ 너무 복잡하고 학술적임, 실무자가 작성하기 어려움

**[OK] 권장 표준**:
```
[OK] 적층 불량 → 와전류 손실 증가 → 국부 과열
```
→ 간결하고 이해하기 쉬움, 실무자가 작성 가능

## S/O/D 평가 근거 작성

### 심각도(S) 평가 근거

**나쁜 예**:
```
고장영향: 철심 소손
S: 9
```

**좋은 예**:
```
고장영향: 철심 소손
S: 9 (변압기 전소 → 교체비 30억원 + 정전손실 50억원 + 복구기간 6개월)
```

**S값 기준표**:
| S | 영향 | 예시 근거 |
|---|------|----------|
| 10 | 인명 피해 | 폭발로 작업자 부상 위험 |
| 9 | 변압기 교체 | 철심 소손 → 교체비 30억 + 손실 50억 |
| 8 | 주요 부품 교체 | 권선 교체 → 비용 10억 + 2개월 |
| 7 | 성능 저하 | 효율 1% 저하 → 연간 3억원 손실 |
| 6 | 수리 필요 | 클램프 교체 → 비용 500만원 + 1주 |
| 5 | 민원 발생 | 소음 증가 → 민원 + 이미지 손상 |

### 발생도(O) 평가 - WebSearch 기반

**나쁜 예**:
```
고장원인: 코팅 두께 부족
O: 5 (추정)
```

**좋은 예**:
```
고장원인: 코팅 두께 부족
O: 4 (CIGRE TB 642: 코팅 불량 1.5%, 0.5-2% 구간 → O=3-4)
```

### 검출도(D) 평가 - 검사 방법 기반

**나쁜 예**:
```
현재검출대책: 육안 검사
D: 6
```

**좋은 예**:
```
현재검출대책: 제작: 육안 검사 (샘플링 5%, 검출률 30% 추정)
D: 7 (수동 샘플링 + 저검출률 -> D=7-8)
```

**D값 기준표**:
| D | 검출 방법 | 예시 |
|---|----------|------|
| 1-2 | 자동 전수 검사 | 실시간 센서 모니터링 (100% 검출) |
| 3-4 | 자동 샘플링 | 10% 자동 샘플링 (검출률 80%) |
| 5-6 | 수동 샘플링 | 5% 육안 검사 (검출률 50%) |
| 7-8 | 간접 추정 | 운전 중 DGA 분석 (검출률 20%) |
| 9-10 | 검출 불가 | 내부 크랙 (파괴 검사만 가능) |

## 예방/검출 대책 차별화

### [X] 중복 대책 (나쁜 예)

```
항목 1: 샘플링 5% → 10%로 증가
항목 2: 샘플링 5% → 10%로 증가
항목 3: 샘플링 5% → 10%로 증가
...
```
**문제**: 46개 항목 모두 동일 대책 = 깊이 없는 사고

### [OK] 차별화된 대책 (좋은 예)

**대책 유형별 분류**:

1. **통계적 공정 관리(SPC)**: 코팅 두께 관리 → X-bar/R 관리도 도입
2. **자동화**: 육안 검사 → AI 비전 검사 시스템
3. **Poka-yoke**: 볼트 누락 → 센서 기반 실수 방지
4. **샘플링 증가**: 저빈도 불량 → 5%→20% 증가
5. **실시간 모니터링**: 온도 관리 → IoT 센서 설치
6. **설비 투자**: 수동 코팅 → 자동 코팅 설비
7. **표준화**: 작업 방법 → 작업 표준서 + 교육
8. **파레토 분석**: 다발 불량 → 중점 관리 항목 선정

**예시 - 층간 절연 파괴 원인별 대책**:

| 고장원인 | 현재대책 | 개선대책 (차별화) |
|----------|----------|-------------------|
| 설계: 적층 설계 오류 | Design Review | **FMEA 사전 검토 + 시뮬레이션** |
| 재료: 코팅 두께 부족 | 샘플링 5% | **자동 코팅 설비 + SPC 관리** |
| 제작: 버(burr) 발생 | 육안 검사 | **레이저 측정 + AI 비전 검사** |
| 시험: 절연 시험 누락 | 시험 절차서 | **바코드 스캔 + 체크리스트** |

## 비용 효과성 평가

### [X] 현실성 없는 대책

```
예방조치: 클린룸 환경 구축 (ISO Class 8 이상)
투자비: 5억원
효과: 습기 제거로 부식 방지
```
**문제**: ROI 분석 없음, 승인 가능성 낮음

### [OK] 비용 효과적 대책

```
예방조치: 제작: 제습기 설치 (습도 <=50%)
투자비: 500만원
효과: 부식 발생률 5%->1% 감소 (연간 손실 3천만원 절감)
ROI: 6개월
```

**ROI 계산 예시**:
```
투자비: 500만원
절감액: 연간 3천만원 (부식 고장 80% 감소)
ROI = 투자비 / 연간 절감액 = 500만 / 3천만 = 0.17년 = 2개월
```
