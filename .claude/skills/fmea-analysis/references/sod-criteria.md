## S/O/D 평가 대상 (CRITICAL!)

> **임채익 이사 (260109)**: "s 심각도는 **영향**에 대해서 평가하는 것이고, o와 d는 **원인**에 대해서 평가하는 거거든요."

| 평가 | 평가 대상 | FMEA 컬럼 | 질문 |
|------|----------|----------|------|
| **S (심각도)** | 고장**영향** | C열 | "이 영향이 얼마나 심각한가?" |
| **O (발생도)** | 고장**원인** | F열 | "이 원인이 얼마나 자주 발생하나?" |
| **D (검출도)** | 고장**원인** | F열 | "이 원인을 얼마나 잘 검출할 수 있나?" |

**핵심**: O/D는 **고장형태(E열)가 아닌 고장원인(F열)**에 대해 평가!

```
[X] 잘못된 평가: "층간단락(형태)이 얼마나 자주 발생하나?"
[OK] 올바른 평가: "절연피막 손상(원인)이 얼마나 자주 발생하나?"
```

## Severity (심각도) - 10단계

> [!] **평가 대상: 고장영향(C열)** - 영향이 얼마나 심각한가?

**고장 영향의 심각도 평가 기준** (1-10):

| S값 | 영향 수준 | 기준 | 예시 |
|-----|----------|------|------|
| **10** | 치명적 | 인명 피해, 화재/폭발, 대규모 정전 | 탱크 폭발, 화재 발생, 송전망 마비 |
| **9** | 매우 심각 | 변압기 전손, 송전 장기 중단 | 권선 소손, 철심 용융, 부싱 폭발 |
| **8** | 심각 | 주요 부품 교체, 장기 정지 (1개월+) | 부싱 파손, 탭절환기 고장, 권선 변형 |
| **7** | 높음 | 계획 외 정지, 주요 수리 (1주+) | 냉각 계통 고장, 대량 누유, 접지 이상 |
| **6** | 중간 | 성능 저하, 수명 단축 | 과열 발생, 효율 저하, 절연 열화 진행 |
| **5** | 낮음 | 정기 점검 시 발견, 예방 조치 필요 | 절연 열화 초기, 소음 증가, 경미한 누유 |
| **4** | 경미 | FAT 불합격, 재작업 필요 | 무부하손 초과, 진동 초과, 시험 불합격 |
| **3** | 미미 | 외관 불량, 터치업 필요 | 도장 불량, 경미한 흠집, 외관 문제 |
| **2** | 거의 없음 | 기능 영향 없음, 미관 문제 | 표식 오류, 라벨 위치, 명판 오류 |
| **1** | 영향 없음 | 발견해도 조치 불필요 | 설계 마진 내 편차 |

### S값 빠른 참조

```
S=10: 화재, 폭발, 인명 피해
S=9:  변압기 전손, 장기 정전
S=8:  주요 부품 교체 (부싱, 탭절환기)
S=7:  계획 외 정지, 수리 필요
S=6:  성능 저하, 수명 단축
S=5:  점검 시 발견, 예방 필요
S=4:  FAT 불합격, 재작업
S=3:  외관 불량
S=2:  미관 문제
S=1:  영향 없음
```

## Detection (검출도) - 10단계

> [!] **평가 대상: 고장원인(F열)** - 원인을 얼마나 쉽게 검출할 수 있는가?

**고장 검출 난이도 평가 기준** (1-10):

| D값 | 검출 능력 | 기준 | 검출 방법 |
|-----|----------|------|----------|
| **10** | 검출 불가 | 운전 중에만 발견, 사전 검출 불가 | 고객 현장 고장, 사고 발생 후 |
| **9** | 거의 불가 | 특수 장비 필요, 검출 매우 어려움 | 고도 진단 장비, 전문가 분석 |
| **8** | 매우 어려움 | 특수 시험 필요, 검출 불확실 | PD 시험, DGA (용존가스분석) |
| **7** | 어려움 | 전문가 판단 필요, 해석 어려움 | SFRA, 열화상, FRA |
| **6** | 보통-어려움 | 형식시험 중 검출 가능 | FAT 상세 시험, 내전압 시험 |
| **5** | 보통 | 표준 시험으로 검출 | 루틴 시험, 무부하 시험, 권선저항 |
| **4** | 보통-쉬움 | 검사로 검출 가능 | 치수 검사, 외관 검사, 토크 확인 |
| **3** | 쉬움 | 육안 검사로 발견 | 시각 검사, 확인 점검 |
| **2** | 매우 쉬움 | 자동 검사, 즉시 발견 | 센서, 자동화 검사, 인터락 |
| **1** | 확실 | 실수 불가능, 자동 방지 | Poka-Yoke, 물리적 방지 |

## RPN 계산 및 기준

### 계산 공식

```python
RPN = S × O × D           # 현재 상태
RPN' = S' × O' × D'       # 개선 후
RPN_감소율 = (RPN - RPN') / RPN × 100  # 개선 효과 (%)
```

### RPN 범위 및 색상 기준

| RPN 범위 | 색상 | 위험도 | 권장 조치 |
|---------|------|--------|----------|
| **300-1000** | 🔴 빨강 | 매우 높음 | **즉시 개선** - 생산 중단 검토 |
| **200-299** | 🔴 빨강 | 높음 | **우선 개선** - 1주 내 대책 수립 |
| **100-199** | 🟡 노랑 | 보통 | **개선 필요** - 1개월 내 대책 수립 |
| **50-99** | 🟢 녹색 | 낮음 | **모니터링** - 정기 점검 시 확인 |
| **1-49** | 🟢 녹색 | 매우 낮음 | **유지** - 현행 유지 |

### RPN 예시

```
예시 1: 고위험
  S=9 (변압기 전손), O=4 (10년 1회), D=7 (SFRA 필요)
  RPN = 9 × 4 × 7 = 252 → 🔴 빨강 (우선 개선)

예시 2: 중위험
  S=6 (성능 저하), O=5 (5년 1회), D=5 (루틴 시험)
  RPN = 6 × 5 × 5 = 150 → 🟡 노랑 (개선 필요)

예시 3: 저위험
  S=4 (FAT 불합격), O=3 (20년 1회), D=3 (육안 검사)
  RPN = 4 × 3 × 3 = 36 → 🟢 녹색 (유지)
```

## AP 결정 규칙

### AIAG-VDA 2019 Action Priority

**AP = H (High) / M (Medium) / L (Low)**

| AP | 조건 | 색상 | 조치 |
|----|------|------|------|
| **H** | S≥9 **OR** (S≥7 AND O≥7) **OR** (S≥7 AND D≥7) | 🔴 빨강 | 필수 개선 |
| **M** | S≥7 **OR** (S≥5 AND O≥5) **OR** (S≥5 AND D≥5) | 🟡 노랑 | 개선 권고 |
| **L** | 나머지 | 🟢 녹색 | 현행 유지 |

### AP 결정 예시

```
예시 1: AP = H
  S=9, O=3, D=5 → S≥9 조건 충족 → H (High)

예시 2: AP = M
  S=7, O=4, D=6 → S≥7 조건 충족 → M (Medium)

예시 3: AP = L
  S=5, O=3, D=4 → 어떤 조건도 불충족 → L (Low)
```

### RPN vs AP 비교

| 항목 | RPN | AP |
|------|-----|-----|
| **같은 값, 다른 위험** | RPN=200 (S=10,O=2,D=10) ≠ (S=4,O=10,D=5) | AP가 S≥9 조건으로 구분 |
| **우선순위 결정** | 수치 비교 (높을수록 위험) | H → M → L 순서 |
| **개선 효과 측정** | RPN 감소율 (%) | AP 단계 변화 (H→M→L) |

**[!] 권장**: RPN과 AP를 **함께 사용**하여 상호 보완

## 변압기 vs 자동차 기준 차이점

| 구분 | 변압기 (일진전기) | 자동차 (현대차) |
|------|-------------------|-----------------|
| **운영 기간** | 30년+ (장기) | 10년 (중기) |
| **생산량** | 연 200대 (소량) | 연 수만 대 (대량) |
| **검출 방식** | FAT/SAT 중심 (출하 전후) | 생산라인 중심 (양산) |
| **S 기준** | 송전 마비 = S10 (사회적) | 인명 피해 = S10 |
| **O 기준** | 1대/년 = O7 | 1000대/년 = O10 |
| **D 기준** | FAT = D5-6 | 공정 검사 = D1-3 |

**핵심**: 변압기는 자동차 대비 **시간 축이 길고, 생산량이 적으며, 검출 방법이 다름**
