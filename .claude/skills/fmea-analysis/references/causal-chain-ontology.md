# 인과관계 체인 온톨로지 (Causal Chain Ontology)

**목적**: 형태→원인, 원인→메커니즘 인과관계 검증 규칙 정의

**적용**: `scripts/validate_causal_chain.py`의 검증 함수에서 파싱

**원리**: SSOT - 이 파일이 인과관계 검증 규칙의 단일 진실 원천

## SECTION:INVALID_COMBINATIONS

> **[BLOCKING]** 아래 조합은 인과관계 불성립! 검증 스크립트에서 자동 차단됨

### MODE_CAUSE:불일치

# 절연 고장형태 <- 전기계산/설계 오류 (인과관계 없음!)
# 층간단락/지락/절연파괴 = 절연 손상 문제
# 턴수/전압비 오류 = 전기 계산 문제 -> 전압비 불일치가 결과임
층간단락: 도장불량, 외관손상, 라벨탈락, 턴수설계오류, 턴수계수오류, 전압비계산오류, 권선수오류, 동선규격오류
지락: 도장불량, 외관손상, 라벨탈락, 턴수설계오류, 턴수계수오류, 전압비계산오류
절연파괴: 도장불량, 외관손상, 라벨탈락, 턴수설계오류, 턴수계수오류, 전압비계산오류
단선: 절연열화, 절연손상, 수분침입 (단선은 도체 문제, 절연 문제 아님)

# 기계적 고장형태 <- 전기적/화학적 원인 (인과관계 없음!)
변형: 절연열화, 수분침입, 오염, 과전압, 부분방전
이완: 절연열화, 과전압, 과전류
탈락: 절연열화, 과전압, 과전류

# 유체 고장형태 <- 전기적 원인 (인과관계 없음!)
누유: 과전압, 과전류, 절연열화
누설: 과전압, 과전류, 절연열화

# 전기계산 고장 <- 절연 원인 (인과관계 없음!)
# 전압비불일치 = 턴수/권선 문제이지 절연 문제가 아님
전압비불일치: 절연열화, 절연손상, 수분침입

### CAUSE_MECHANISM:불일치
진동: 열분해, 가수분해, 산화분해
수분침입: 피로파괴, 크리프, 공진
과전압: 마모진행, 부식진행, 크리프
턴수오류: 절연파괴, 절연저항감소, 부분방전

## 파싱 규칙

- `## SECTION:<섹션명>` 형식으로 섹션 구분
- `### CATEGORY:<카테고리명>` 또는 `### STAGE:<단계명>` 형식으로 하위 구분
- `고장형태:`, `유효원인:`, `원인:`, `유효메커니즘:` 등 키로 목록 정의
- 쉼표(,)로 키워드 구분
