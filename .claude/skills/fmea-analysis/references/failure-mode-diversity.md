# 고장형태 다양화 전략 (CRITICAL!)

## 왜 다양화가 필요한가?

**잘못된 예시** (실제 규소강판 FMEA 문제):
```
기능: 자속 유도 (13개 항목)
  └─ 고장형태: "과열" 13개 ← [X] 단일 형태만 사용!
```

**올바른 예시** (구체적 현상 다양화):
```
기능: 자속 유도 (16개 항목)
  ├─ 고장형태: "층간단락" 4개
  ├─ 고장형태: "과열" 4개
  ├─ 고장형태: "부식" 3개
  ├─ 고장형태: "변형" 3개
  └─ 고장형태: "절연열화" 2개
```

## 고장형태 다양화 방법

**Step 1: 기능 분석 → 가능한 모든 고장 형태 식별**

| 기능 | 가능한 고장형태 | 실제 시나리오 |
|------|----------------|--------------|
| 자속 유도 | **층간단락**: 철심 층간 절연 파괴 | 절연 코팅 손상, 버 발생 |
| 자속 유도 | **과열**: 국부 온도 상승 | 와전류 손실 증가, 냉각 불량 |
| 자속 유도 | **변형**: 철심 형상 변화 | 진동, 외력, 조립 오류 |
| 자속 유도 | **부식**: 표면 산화/열화 | 습기 침투, 환경 영향 |

**Step 2: 각 고장형태별 독립적인 Failure Chain 구축**

**예시: 자속전도 손실 최소화**

```
고장영향: 철손 증가
├─ 고장형태 1: "층간단락"
│   ├─ 원인: 재료: 절연 코팅 두께 부족
│   ├─ 원인: 제작: 버(burr) 발생
│   └─ 원인: 설계: 절연 코팅 설계 오류
│
├─ 고장형태 2: "과열"
│   ├─ 원인: 재료: 규소강판 전기 저항 부족
│   ├─ 원인: 제작: 적층 불량
│   └─ 원인: 운영: 냉각 불량
│
└─ 고장형태 3: "부식"
    ├─ 원인: 설계: 방청 처리 설계 미흡
    └─ 원인: 제작: 표면 처리 불량
```

## AIAG-VDA 8가지 분류와의 관계

**참조용으로만 사용** (회의 합의):

| AIAG-VDA 분류 | 해당 구체적 고장형태 예시 |
|---------------|--------------------------|
| 부재 (Loss) | 단선, 탈락, 누락 |
| 부족 (Partial) | 두께 부족, 강도 부족 |
| 과도 (Excessive) | 과열, 과전압, 과전류 |
| 간헐 (Intermittent) | 간헐 접촉불량, 진동 |
| 의도하지않음 (Unintended) | 누설전류, 누유 |
| 저하 (Degraded) | 열화, 마모, 부식 |
| 지연 (Delayed) | 응답 지연, 동작 지연 |
| 역효과 (Inverse) | 역극성, 역방향 흐름 |

## Transformer Core 실제 고장 메커니즘

**WebSearch 기반 실제 사례** ([ResearchGate 2024](https://www.researchgate.net/publication/362096767_Experimental_investigation_on_impacts_of_insulation_damage_fault_between_laminations_of_power_transformers)):

| 고장형태 | 물리적 메커니즘 | 출처 |
|----------|----------------|------|
| **층간단락** | 층간 절연 손상 → 층간 전류 → 와전류 폭증 → 국부 과열 | ResearchGate, IEEE C57.125 |
| **과열** | 와전류 증가 → 발열 → 온도 상승 → 절연 열화 | IEC 60076-2 |
| **변형** | 진동/외력 → 철심 변형 → 자기 특성 변화 | CIGRE TB 642 |
| **부식** | 습기 침투 → 산화 → 부식층 형성 → 자기 특성 저하 | IEC 60076-1 |
| **절연열화** | 온도 순환 → 절연 코팅 균열 → 절연 저항 저하 | DOE 2024 Report |
| **진동** | 적층 불량 → 공극 → 자왜 현상 → 소음/진동 | Transformer Magazine |

**중요**: 위 고장형태들은 **독립적인 failure chain**이며, 각각 별도로 평가해야 함!
