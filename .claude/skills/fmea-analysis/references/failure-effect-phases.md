## 핵심 원칙

### 고장영향 = 기능 실패의 결과 (CRITICAL!)

**AIAG-VDA 정의**:
> 고장영향(Failure Effect)은 고장형태가 발생했을 때 **기능 실패로 인해 나타나는 결과**

### 인과관계 필수

```
기능 → 기능이상(고장형태) → 고장영향
       (현재 관찰 가능)      (미래 결과)
```

**올바른 예**:
- 기능: 자속을 전달한다 → 고장영향: 전압 변환 불가 (인과관계 O)
- 기능: 절연을 유지한다 → 고장영향: 지락사고 (인과관계 O)

**잘못된 예**:
- 기능: 자속을 전달한다 → 고장영향: 적층 불량 (인과관계 X - 이건 고장형태!)
- 기능: 권선을 지지한다 → 고장영향: FAT 불합격 (인과관계 X - 발견 장소일 뿐!)

## 고장영향 정의

### C열 고장영향이란?

| 항목 | 정의 |
|------|------|
| **정의** | 기능이 실패했을 때 **미래에 발생하는 결과** |
| **질문** | "이 고장형태가 발생하면 **앞으로 무슨 일**이 생기나?" |
| **시점** | 미래 (고장형태 발생 이후) |
| **관점** | 최종 사용자/고객이 경험하는 결과 |

### 고장영향 vs 발견 시점 (혼동 금지!)

| 구분 | 고장영향 (C열) | 발견 시점 |
|------|---------------|----------|
| **정의** | 기능 실패의 결과 | 문제를 발견하는 장소/단계 |
| **반영 위치** | C열 (고장영향) | S값 또는 J열 (검출대책) |
| **예시** | 전압 변환 불가, 과열, 지락사고 | 제작 중, 시험 중, 운전 중 |

### 잘못된 고장영향 예시

| 잘못된 C열 값 | 문제점 | 올바른 처리 |
|--------------|--------|------------|
| "조립 불합격" | 발견 장소(제작 중)를 영향으로 작성 | S=4로 반영, C열: 기능 관련 결과 |
| "FAT 불합격" | 발견 장소(시험 중)를 영향으로 작성 | S=6로 반영, C열: 기능 관련 결과 |
| "용접 불량" | 고장형태를 영향으로 작성 | E열로 이동 |
| "치수 초과" | 고장형태를 영향으로 작성 | E열로 이동 |

## S값으로 심각도 반영

### S값 범위 가이드

발견/발생 시점에 따른 심각도는 **S값**으로 반영합니다.

| S값 범위 | 심각도 | 발견/발생 시점 | 비용 영향 |
|---------|--------|---------------|----------|
| **2-5** | 경미 | 제작 중 발견 가능 | 낮음 (재작업) |
| **4-7** | 중간 | 시험 중 발견 가능 | 중간 (일정 지연) |
| **7-10** | 심각 | 운전 중 발생 | 높음 (현장 고장) |

### S값 분포 품질 기준

FMEA 품질을 위해 다양한 S값 범위 확보:

| S값 범위 | 권장 비율 | 의미 |
|---------|----------|------|
| 2-5 | ≥20% | 조기 발견 시 경미한 영향 |
| 6-7 | ≥20% | 시험/검사로 발견 가능 |
| 8-10 | ≥30% | 운전 중 심각한 영향 |

### S값 결정 기준

| S값 | 심각도 | 기준 |
|-----|--------|------|
| 10 | 위험 | 안전 사고, 인명 피해 |
| 9 | 심각 | 제품 완전 고장, 화재/폭발 |
| 8 | 높음 | 주요 기능 상실, 장기 정전 |
| 7 | 보통-높음 | 성능 저하, 단기 정전 |
| 6 | 보통 | 일부 기능 저하, 시험 불합격 |
| 5 | 낮음-보통 | 외관/소음 문제, 재시험 |
| 4 | 낮음 | 재작업 필요 |
| 3 | 경미 | 수정 가능 |
| 2 | 매우 경미 | 사소한 불편 |
| 1 | 없음 | 영향 없음 |

## 기능별 고장영향 도출

### 도출 프로세스

```
1. 기능 정의
   ↓
2. 기능이 실패하면? (질문)
   ↓
3. 최종 사용자에게 미치는 결과 (답변)
   ↓
4. 고장영향 작성 + S값 결정
```

### 기능-고장영향 매핑 템플릿

| 기능 유형 | 질문 | 고장영향 후보 |
|----------|------|-------------|
| **전달 기능** | 전달이 안 되면? | 기능 상실, 성능 저하 |
| **유지 기능** | 유지가 안 되면? | 누출, 파괴, 사고 |
| **지지 기능** | 지지가 안 되면? | 변형, 변위, 손상 |
| **보호 기능** | 보호가 안 되면? | 오염, 손상, 사고 |
| **조절 기능** | 조절이 안 되면? | 과/부족, 불안정 |

### 과/부족/유해 관점 적용

각 기능에 대해 3가지 관점으로 고장영향 검토:

| 관점 | 질문 (가이드 V1.3) | 고장영향 예시 |
|------|------|-------------|
| **부족** | 기능 결과물이 부족하면? | 성능 저하, 기능 상실 |
| **과** | 기능 결과물이 과하면? | 과열, 과부하, 손상 |
| **유해** | 기능 결과물 정상인데 부작용? | 누출, 오염, 간섭 |

## 검증 체크리스트

### FMEA 작성 후 자가 검증

- [ ] **인과관계 검증**: 모든 고장영향이 해당 기능 실패의 결과인가?
- [ ] **발견 장소 제거**: "조립 불합격", "FAT 불합격" 같은 발견 장소가 C열에 없는가?
- [ ] **고장형태 구분**: E열 값이 C열에 잘못 들어간 것은 없는가?
- [ ] **S값 범위 확인**: S값이 고장영향의 심각도를 적절히 반영하는가?
- [ ] **S값 분포 확인**: 다양한 S값 범위가 포함되어 있는가?

### 인과관계 검증 예시

| 기능 | 고장영향 | 인과관계 | 판정 |
|------|---------|---------|------|
| 자속 전달 | 전압 변환 불가 | 자속 전달 실패 → 전압 변환 불가 | O |
| 자속 전달 | 적층 불량 | 자속 전달 실패 → 적층 불량? | X (역순) |
| 절연 유지 | 지락사고 | 절연 실패 → 지락사고 | O |
| 절연 유지 | FAT 불합격 | 절연 실패 → FAT 불합격? | X (발견 장소) |

## WebSearch 연계 가이드

### 고장영향 검색 키워드

**목적**: 기능 실패로 인한 결과 사례 검색

**추천 검색어**:
```
WebSearch: "transformer {부품명} failure consequence effect"
WebSearch: "IEC 60076 {부품명} failure impact customer"
WebSearch: "CIGRE transformer {기능} loss effect"
```

**검색 결과 활용**:
- "ground fault" → 지락사고 (S=10)
- "winding burnout" → 권선 소손 (S=9)
- "voltage regulation failure" → 전압 조정 불가 (S=7)
- "efficiency loss" → 효율 저하 (S=5)
