# 일진전기 조직 구조 및 팀명

**FMEA 작성 시 반드시 이 팀명 사용**

---

## 조직 구조

```
일진전기
├─ 전기설계팀 (Electrical Design Team)
│   - 전기 회로 설계, 권선 설계, 절연 설계
│   - 담당: 권선, 절연시스템, 전기 설계
├─ 구조설계팀 (Mechanical Design Team)
│   - 기계 구조 설계, 냉각 시스템, 탱크 설계
│   - 담당: 철심, 클램프, 리드, 절연시스템, 부스바, 지지 구조
├─ 개발팀 (R&D Team)
│   - 신기술 개발, 프로토타입 제작, 시험 설계
│   - 담당: 신제품 개발, 기술 혁신
├─ 생산팀 (Production Team)
│   - 제조, 조립, 생산 공정 관리
│   - 담당: 제조 공정, 조립, 압착, 용접
├─ 생산기획팀 (Production Planning Team)
│   - 생산 계획, 일정 관리, 자재 소요 계획
│   - 담당: 생산 일정, MRP, 공정 개선
├─ QA팀 (Quality Assurance Team)
│   - 품질 보증, 시험, 검사, 인증
│   - 담당: 검사, 시험, 품질 관리, 인증
├─ 시공팀 (Installation Team)
│   - 현장 설치, 시운전, 유지보수
│   - 담당: 현장 설치, 시운전, AS
├─ 구매팀 (Procurement Team)
│   - 자재 구매, 공급사 관리, 품질 인증
│   - 담당: 자재 구매, 공급사 선정 및 관리
├─ 프로젝트관리팀 (Project Management Team)
│   - 프로젝트 관리, 일정 조율, 고객 대응
│   - 담당: 프로젝트 전체 관리, 고객 대응
└─ 총무팀 (General Affairs Team)
    - 총무, 인사, 재무, 법무
    - 담당: 행정 지원, 계약 관리
```

---

## 담당자 매핑 규칙

### 설계 관련

- 전기 설계 → 전기설계팀
- 구조/기계 설계 → 구조설계팀
- 신기술/개발 → 개발팀

### 생산 관련

- 제조/조립/용접/압착 → 생산팀
- 생산 일정/공정 개선 → 생산기획팀

### 품질 관련

- 시험/검사/품질 관리 → QA팀

### 자재 관련

- 재료/부품 구매 → 구매팀
- 공급사 품질 인증 → QA팀 + 구매팀

### 설치/유지보수

- 현장 설치/시운전 → 시공팀

### 프로젝트 관리

- 전체 일정 관리 → 프로젝트관리팀

---

## 담당자 표기 형식

```
단일 팀: "전기설계팀"
복수 팀: "전기설계팀/QA팀"
협업 팀: "구조설계팀/생산팀/QA팀"
```

## 잘못된 팀명 (절대 사용 금지)

- [X] 설계팀 → [OK] 전기설계팀 또는 구조설계팀
- [X] 시험팀 → [OK] QA팀
- [X] 품질팀 → [OK] QA팀
- [X] 재료팀 → [OK] 구매팀 또는 개발팀
- [X] 조립팀 → [OK] 생산팀
