## [TARGET] 핵심 개선사항 (2025-11-13 교육 반영)

### 1. [!] CRITICAL: 체계적 준비의 중요성 (20년 시행착오 교훈)

**현대자동차의 20년 시행착오 교훈에서 배운 핵심 원칙**:

- **급하게 FMEA 작성 시작 금지**: 충분한 준비 없이 작성하면 품질 저하 및 재작업 발생
- **체계적 준비 필수**: 부품 분석, 과거 문제 검토, 노이즈 인자 파악 후 작성 시작
- **과거 문제 DB 우선 활용**: 유사 부품의 과거 문제 60% 재활용 가능

**방법 1 (간소화 1시트)을 위한 체계적 준비** (3단계 워크플로우):

**1단계: 준비 및 분석** (충분한 정보 수집)

- WebSearch 기반 기술 정보 수집 → 자연스럽게 항목 수 결정 (품질 우선, 개수 제한 없음)
- 과거 문제 DB 조회 (60% 재활용 목표)
- 5대 노이즈 인자 검토
- 주변 시스템/부품 파악

**2단계: FMEA 작성** (체계적 분석 기반 작성)

- 구체적 고장형태 사용 (변형, 크랙, 과열 등)
- 4단계 라이프사이클 반영 (설계/재료/제작/시험)
- S/O/D/AP 평가 (변압기 기준)

**3단계: 검증 및 완료** (품질 확인)

- WebSearch 기반 항목 생성 완료 (품질 우선, 개수 제한 없음)
- 품질 체크리스트 검증
- H 등급 고장 대책 완료

**핵심 교훈**: "준비 없이 작성 시작 금지" (급하게 작성 → 품질 저하 → 재작업 반복)

### 3. [DATA] 과거 문제 DB - 핵심 자산 (60% 재활용 목표)

**과거 문제 DB의 중요성**:

- **현대자동차 사례**: 20년간 축적된 부품별 마스터 파일
- **재활용률**: 60% 이상 (신규 FMEA 작성 시간 70% 단축)
- **핵심 자산**: 회사의 고장 예방 노하우 집약체
- **일진전기 필요성**: 변압기 수명 30~40년 → 과거 데이터 축적 필수

**DB 구축 전략** (3단계):

#### Phase 1: 초기 구축 (6개월~1년)
```
목표: 주요 부품 10종 마스터 파일 생성

우선순위 부품:
1. 철심 (CORE)
2. 권선 (WINDING)
3. 부싱 (BUSHING)
4. 탭 절환기 (OLTC)
5. 절연유 시스템
6. 냉각 시스템
7. 가스켓/씰링
8. 접지 시스템
9. 부스바/리드선
10. 보호 계전기

방법:
- 과거 AS 데이터 수집 (5년치)
- 시험 불합격 이력 수집
- 고객 클레임 데이터 수집
- 엑셀로 정리 (현대차 양식 활용)
```

#### Phase 2: 확장 및 정제 (1~2년)
```
목표: 모든 부품 마스터 파일 + 노이즈 인자 표준 테이블

확장 부품:
- 2차 부품 (센서, 밸브, 배관 등)
- 보조 기기 (팬, 펌프, 온도계 등)

노이즈 인자 표준:
- 5대 노이즈별 세부 항목 정리
- 부품별 특화 노이즈 추가
- 계절/지역별 특성 반영

데이터 품질 관리:
- 중복 제거
- 통합 (유사 문제 통합)
- 표준 용어로 정리
```

#### Phase 3: 고도화 (2년 이후)
```
목표: 지능형 DB 시스템

고도화 방향:
- 검색 기능 강화 (키워드 검색)
- 통계 분석 (고장 트렌드)
- AI 연계 (유사 문제 자동 추천)
- 시각화 (대시보드)

유지보수:
- 분기별 업데이트
- 신규 문제 즉시 등록
- 해결 방안 지속 보완
```

**과거 문제 DB 위치**:
```
04.HMS_DB/QA자료/01_원본데이터/QA_품질이력.db (SQLite)
```

**DB 테이블 구조**:
- 부품별 품질 이력 (철심, 권선, 부싱 등)
- 문제점, 원인, 조치내용, 발생시기, 고객사

### 5. [TIP] 실습 예제 추가 (교육자료 기반)

#### 예제 2: 종이컵 FMEA (입문용)

**목표**: FMEA 방법론을 간단한 제품으로 이해

**3단계: 초점 요소 선정**
| NO | 상위 수준 | 초점 요소 | 하위 수준 | FMEA 수행 대상 |
|----|----------|----------|----------|---------------|
| 1 |  | 종이컵 |  | ● |
| 2 |  | 종이컵 | 컵 본체 (원통) |  |
| 3 |  | 종이컵 | 접합부 (시임) |  |
| 4 |  | 종이컵 | 바닥 (원형) |  |
| 5 | 주변 부품 | 뚜껑 |  |  |

**5단계: 기능 분류표**
| 초점 요소 | 기능 | 정량적 요구사항 | 우선순위 | 고장 형태 |
|----------|------|----------------|---------|----------|
| 종이컵 | 음료 담기 | 250ml 용량 유지 | 1 | ● 누수 (접합부 불량), ● 균열 (측면 파손) |
| 종이컵 | 견고성 | 500g 하중 견딤 | 2 | ● 변형 (찌그러짐), ● 파손 (바닥 분리) |

**FMEA 결과** (일부 발췌)

| 고장 형태 | 고장 원인 | S | 현재 예방 대책 | O | 현재 검출 대책 | D | AP | 예방 조치 | 검출 조치 | S' | O' | D' | AP' |
|----------|----------|---|--------------|---|--------------|---|----|---------|---------|----|----|----|-----|
| 누수 | 접합부 접착 불량 | 7 | PE 코팅 2회 도포 | 5 | 육안 검사 | 6 | M | PE 코팅 3회, 초음파 검사 | 수압 시험 추가 | 7 | 2 | 2 | L |
| 변형 | 종이 두께 부족 | 6 | 250g/m² 원지 사용 | 5 | 원지 입고 검사 | 2 | M | 280g/m² 원지로 변경 | 입고 시 두께 전수 검사 | 6 | 2 | 1 | L |

**개선 결과**:

- 접합부 누수: S=7, O=5→2, D=6→2, AP=M→L (60% 개선)
- 종이 두께: S=6, O=5→2, D=2→1, AP=M→L (60% 개선)

**학습 포인트**:

- 간단한 제품도 체계적 분석 가능
- FCM 체인 이해: 접착 불량 (원인) → PE 코팅 부족 (메커니즘) → 누수 (모드)
- 예방 vs 검출 대책 구분

#### 예제 3: 자동차 콤비네이션 스위치 FMEA (복잡한 전기 부품)

**개요**:

- **부품명**: 콤비네이션 스위치 (방향지시등 + 와이퍼 + 전조등 통합)
- **부품 수**: 15개 (하우징, 레버, 단자, 스프링 등)
- **재질 수**: 7종 (플라스틱, 구리, 은-니켈 합금 등)
- **난이도**: 중급 (전기 접점 + 기계 구조 복합)

**핵심 고장 사례**: 단자 부식

**FCM 분석**:
```
Failure Cause (원인):
- 단자 재질이 순동(Cu)으로 부식에 취약
- 염분 환경 (해안가, 제설제) 노출

Failure Mechanism (메커니즘):
- 단자 표면 산화 → 접촉 저항 증가 → 발열 → 가속 부식

Failure Mode (고장 형태):
- 간헐적 작동 불량 → 완전 작동 불능

Failure Effect (영향):
- 방향지시등 작동 불가 → 교통사고 위험
```

**FMEA 결과** (단자 부식 케이스):

| 항목 | 내용 | 점수 |
|------|------|------|
| **고장 영향** | 방향지시등 작동 불가 → 교통사고 위험 (도로교통법 위반) | S=9 |
| **현재 예방 대책** | 순동(Cu) 단자 사용, 하우징 밀봉 | O=5 |
| **현재 검출 대책** | 육안 검사, 작동 시험 | D=5 |
| **Action Priority** | **H (High)** - 즉시 대책 필요 | |
| **예방 조치** | 단자 재질 변경: Cu → Ag-Ni 합금 (부식 저항 10배) | O'=3 |
| **검출 조치** | 염수 분무 시험 추가 (96시간), 접촉 저항 측정 | D'=1 |
| **개선 후** | S=9 (불변), O=3, D=1, **AP=L** | |
| **리스크 감소율** | **88% 감소** (RPN 기준: 225→27) | |

**비용 분석**:

- 단자 재질 변경 비용: +$0.50/개
- 사고 배상 비용 (1건): $50,000+
- ROI: 1건 사고 예방 시 100,000개 이상 생산 가능

**학습 포인트**:

- 전기 부품의 FCM 분석: 재질 → 메커니즘 → 모드
- 비용-효과 분석의 중요성
- 단순 부품도 심각도 S=9 가능 (안전 관련)

**변압기 적용**:

- 유사 사례: 부스바 단자, 압착 단자
- 동일 메커니즘: 부식 → 접촉 저항 → 발열 → 절연 파괴
- 대책: 은 도금, 밀봉 구조, 정기 점검

### 6. 🧩 노이즈 인자 5가지 표준 (변압기 특화)

**H/W 부품 5대 노이즈 인자** (현대차 표준을 변압기에 적용):

#### 1. 외부 환경 노이즈
**정의**: 변압기가 설치/운영되는 환경 조건

| 노이즈 항목 | 변압기 특화 내용 | 예시 |
|-----------|---------------|------|
| **온도** | 극한 온도: -40°C (겨울, 북부 지역) ~ +50°C (여름, 옥외 설치) | 절연유 점도 변화, 가스켓 경화/연화 |
| **습도** | 고습도: 95% 이상 (해안가, 장마철) | 부싱 표면 섬락, 접속부 부식 |
| **진동** | 지진 (0.3g 이상), 교통 진동 | 볼트 풀림, 권선 변형 |
| **충격** | 운송 중 충격 (3g 이상) | 철심 손상, 부싱 균열 |
| **염분** | 해안가 염해 (5km 이내) | 부스바 부식, 탱크 부식 |
| **낙뢰** | 뇌 임펄스 전압 (BIL 650kV/154kV) | 부싱 섬락, 권선 절연 파괴 |
| **먼지/오염** | 산업 지역 분진, 조류 배설물 | 부싱 오염, 냉각 핀 막힘 |

#### 2. 계통/고객 사용 노이즈 (자동차의 "고객 사용"에 해당)
**정의**: 전력 계통 조건 및 운영 패턴

| 노이즈 항목 | 변압기 특화 내용 | 예시 |
|-----------|---------------|------|
| **과부하** | 정격 용량 초과 (110~150%) | 권선 과열, 절연 열화 가속 |
| **부하 변동** | 급격한 부하 변동 (±30% 이상) | 열 사이클 스트레스, 볼트 풀림 |
| **전압 변동** | 계통 전압 ±10% 변동 | 무부하 손실 변화, 소음 증가 |
| **고조파** | 5차, 7차 고조파 (THD 5% 이상) | 와전류 손실 증가, 국부 과열 |
| **단시간 과전류** | 계통 단락 시 수천~수만 A | 권선 전자기력, 지지 구조 파손 |
| **빈번한 개폐** | 무부하 투입 돌입 전류 (정격의 8~10배) | 권선 진동, 기계적 피로 |

#### 3. 시스템 상호작용 노이즈 (자동차의 "시스템 상호작용"과 동일)
**정의**: 변압기 내부 또는 주변 기기와의 상호작용

| 노이즈 항목 | 변압기 특화 내용 | 예시 |
|-----------|---------------|------|
| **계통 단락 사고** | 2차측 단락 전류 (정격의 20배 이상) | 권선 변형, 부스바 용융 |
| **공진** | 5차/7차 고조파 공진 | 소음 증가, 진동 증폭 |
| **전자기 간섭** | 권선-철심 간 누설 자속 | 탱크 국부 과열 (와전류) |
| **무효 전력 변동** | 경부하 시 무효 전력 발생 | 전압 상승, 계통 불안정 |
| **타 변압기 영향** | 병렬 운전 시 순환 전류 | 불평형 전류, 과열 |

#### 4. 시간 경과에 따른 변화 노이즈 (자동차와 동일)
**정의**: 30~40년 운영 기간 동안의 경년 열화

| 노이즈 항목 | 변압기 특화 내용 | 예시 |
|-----------|---------------|------|
| **절연유 열화** | 산가(Acid Number) 증가, 수분 증가 | 절연 내력 저하, 슬러지 생성 |
| **절연지 열화** | 중합도(DP) 저하 (신품 1200 → 250 이하 수명) | 기계적 강도 저하, 찢어짐 |
| **철심 부식** | 절연 코팅 손상 → 부식 → 와전류 증가 | 철손 증가, 온도 상승 |
| **가스켓 열화** | 고무 경화, 압축영구변형 | 절연유 누유 (연간 1~10L) |
| **부싱 열화** | 내부 절연 열화, 크랙 발생 | 부분 방전, 섬락 위험 |
| **권선 피로** | 열 사이클 반복 (일 2회 이상) | 권선 변형, 단선 |

#### 5. 부품 간 변동 노이즈 (자동차와 동일)
**정의**: 제조 공차, 조립 오차, 재료 편차

| 노이즈 항목 | 변압기 특화 내용 | 예시 |
|-----------|---------------|------|
| **철심 재질 편차** | 규소강판 자속 밀도 ±5% | 철손 편차, 소음 편차 |
| **권선 치수 공차** | 권선 높이 ±2mm | 절연 거리 부족, 부분 방전 |
| **절연유 품질 편차** | 절연 내력 ±10kV/mm | 절연 파괴 전압 변동 |
| **조립 오차** | 볼트 토크 ±10% | 압착력 부족, 접촉 저항 증가 |
| **용접 품질 편차** | 용접 강도 ±20% | 접합부 균열, 누유 |
| **공급사 부품 편차** | 부싱 절연 내력 ±5% | 섬락 위험 증가 |

| **ChatGPT** | 고장 원인 브레인스토밍 | 아이디어 30~50개 도출 | 변압기 특화 지식 부족 | 7단계 고장 원인 분석 |
| **Claude (Sonnet)** | 문서 분석, 기술 자료 요약 | 긴 문서 빠르게 이해 | 실시간 검색 불가 | 참고 자료 리뷰 시 |
| **MidJourney/DALL-E** | Boundary Diagram 초안 | 시각화 아이디어 | 정확도 낮음, 수정 필수 | 4단계 다이어그램 작성 |
| **Copilot (Excel)** | FMEA 표 자동 생성 | 반복 작업 자동화 | 수식 오류 가능 | 8단계 데이터 입력 시 |

#### Notebook LM 활용 사례 (상세)

**Step 1: 소스 업로드**
```
업로드 자료:
1. 과거 FMEA 파일 10개 (PDF, Excel)
2. 일진전기 설계 표준 (PDF)
3. IEC 60076 표준 요약 (PDF)
4. 과거 AS 보고서 20개 (PDF)
5. 시험 성적서 샘플 (PDF)

→ Notebook LM이 자동 인덱싱 (5분 소요)
```

**Step 2: 질문 예시**
```
Q1: "초고압 변압기 철심에서 발생 가능한 고장 형태 10가지는?"
→ AI 답변: 철심 과열, 와전류 증가, 접지 불량, 부식, 소음 증가... (출처 표시)

Q2: "철심 과열의 고장 원인과 메커니즘을 FCM 형식으로 설명해줘"
→ AI 답변:
   Cause: 철심 재질 불량, 적층 접착 불량
   Mechanism: 와전류 손실 증가 → 국부 과열 → 절연 코팅 손상
   Mode: 철손 증가, 온도 상승
   Effect: 절연유 열화, 냉각 시스템 부담

Q3: "과거 AS 데이터에서 철심 관련 문제가 몇 건이고, 주요 원인은?"
→ AI 답변: 총 7건, 주요 원인은 접지 불량(3건), 재질 불량(2건), 운송 손상(2건)
```

**Step 3: FMEA 초안 생성**
```
프롬프트:
"초고압 변압기 철심 FMEA를 현대차 9단계 방식으로 작성해줘.
과거 AS 데이터와 IEC 표준을 참고하고, S/O/D 점수도 제시해."

→ AI 초안 생성 (30분 소요, 사람은 8시간 소요)
→ 엔지니어 검토 및 수정 (2시간)
→ 총 작업 시간: 2.5시간 (기존 8시간 → 70% 단축)
```

**주의사항**:

- AI 결과는 **반드시 엔지니어가 검증** (책임은 엔지니어)
- S/O/D 점수는 **AI 제안 → 워크샵에서 합의** 필요
- 고객사 제출 시 "AI 활용" 여부 확인 (일부 고객사는 금지)
