## CRITICAL: WebSearch 필수 원칙

**[X] 잘못된 접근**:
- WebSearch는 했지만 통계 데이터를 O/D 값에 반영하지 않음
- "CIGRE TB 642: 고장률 0.3% 발견" → 그냥 O=5로 임의 설정

**[OK] 올바른 접근**:
- WebSearch로 찾은 통계를 O/D 매핑 테이블로 변환
- "CIGRE TB 642: 고장률 0.3% 발견" → 매핑 테이블 참조 → O=2 (0.5% 미만)

## O(발생도) 값 산정 기준

### 매핑 테이블 1: 고장률 → O 값

| 고장률 범위 | O 값 | 근거 작성 예시 | 신뢰도 |
|------------|------|---------------|--------|
| >10% | 9-10 | "CIGRE TB 642: 절연 고장 12% (10-15% 구간)" | 높음 |
| 5-10% | 7-8 | "IEEE C57.125: 권선 변형 8% (5-10% 구간)" | 높음 |
| 2-5% | 5-6 | "CIGRE: 철심 고장 3.2% (2-5% 구간)" | 높음 |
| 0.5-2% | 3-4 | "통계: 코팅 불량 1.5% (0.5-2% 구간)" | 중간 |
| 0.1-0.5% | 2-3 | "IEEE: 희귀 고장 0.3% (0.1-0.5% 구간)" | 중간 |
| <0.1% | 1-2 | "매우 희귀 고장 <0.1%" | 낮음 |

|---------------------|------|---------------|

| "흔함", "자주 발생" | 7-8 | "자주 발생하는 고장 원인" |
| "가끔 발생", "드물지 않음" | 5-6 | "제조 공정에서 가끔 발생" |
| "드물게 발생" | 3-4 | "과거 사례 드물게 보고됨" |
| "매우 드물게", "희귀" | 2-3 | "희귀 고장 형태" |
| "거의 없음", "이론적" | 1-2 | "이론적으로만 가능" |

### WebSearch 출처별 신뢰도

**최고 신뢰도** (O 값 ±0):
- CIGRE Technical Brochure (TB) - 전력설비 통계 표준
- IEEE C57 시리즈 - 변압기 표준
- IEC 60076 - 국제 표준

**높은 신뢰도** (O 값 ±1):
- Transformer Magazine 사례 연구
- 제조사 기술 문서
- 대학/연구소 논문

**중간 신뢰도** (O 값 ±2):
- 일반 기술 자료
- 산업 통계 (출처 불명확)
- 전문가 의견

**주의 필요** (사용 시 명시):
- 블로그/포럼 의견
- 오래된 자료 (>10년)
- 출처 없는 통계

## WebSearch → O/D 값 산정 워크플로우

### Step 1: WebSearch 수행 (5개 출처)

```
1. IEC 60076: 설계 기준, 시험 방법
2. IEEE C57: 고장 분석, 통계
3. CIGRE TB: 고장률 통계 [*] (O 값 산정 핵심)
4. Transformer Magazine: 실제 사례
5. 일반 검색: 보충 정보
```

### Step 2: 통계 데이터 추출

**O 값 산정용 정보 찾기**:
- 고장률: "X% of transformers experience Y failure"
- 발생 빈도: "commonly", "rarely", "occasionally"
- 사례 수: "100 cases out of 10,000 units"

**D 값 산정용 정보 찾기**:
- 검출 방법: "visual inspection", "automated testing"
- 샘플링률: "10% sampling", "100% inspection"
- 검출률: "detection rate of 80%"

### Step 3: 매핑 테이블 적용

**예시 1: CIGRE TB 642에서 "코팅 불량 1.5%" 발견**

1. 통계: 1.5%
2. 매핑 테이블 1 참조: 0.5-2% → O=3-4
3. 신뢰도: CIGRE (최고) → O=4 (구간 중간값)
4. 근거 작성: "O=4 (CIGRE TB 642: 코팅 불량 1.5%, 0.5-2% 구간)"

**예시 2: IEEE에서 "육안 검사 5% 샘플링" 언급**

1. 검출 방법: 육안 샘플링 5%
2. 매핑 테이블 3 참조: 수동 샘플링 5-10%, 검출률 30-50% → D=5-6
3. 일반적 D 값: D=6
4. 근거 작성: "D=6 (육안 샘플링 5%, 검출률 30% 추정)"

### Step 4: 근거 문서화

**필수 포함 요소**:
1. 출처: "CIGRE TB 642", "IEEE C57.125"
2. 통계: "1.5%", "검출률 30%"
3. 구간: "(0.5-2% 구간)", "(D=5-6 구간)"
4. 최종 값: "O=4", "D=6"

**근거 작성 템플릿**:
```
O=[값] ([출처]: [통계], [구간])
D=[값] ([검출방법], 검출률 [%] 추정)
```

## 실전 예시

### 예시 1: 층간 절연 파괴

**WebSearch 결과**:
- CIGRE TB 642: "Core lamination insulation failure: 2.3%"
- IEEE C57.152: "Visual inspection with 10% sampling"

**O 값 산정**:
1. 고장률: 2.3%
2. 매핑: 2-5% 구간 → O=5-6
3. CIGRE (최고 신뢰도) → O=6 (구간 상단, 안전 여유)
4. **근거**: `O=6 (CIGRE TB 642: 절연 고장 2.3%, 2-5% 구간)`

**D 값 산정**:
1. 검출 방법: 육안 검사 10% 샘플링
2. 매핑: 샘플링 5-10%, 검출률 30-50% → D=5-6
3. 일반적 → D=6
4. **근거**: `D=6 (육안 샘플링 10%, 검출률 40% 추정)`

### 예시 2: 와전류 손실

**WebSearch 결과**:
- IEC 60076-1: "Core loss test required for all units"
- Transformer Magazine: "Core loss anomaly detected in 0.8% of cases"

**O 값 산정**:
1. 고장률: 0.8%
2. 매핑: 0.5-2% 구간 → O=3-4
3. Transformer Magazine (높은 신뢰도) → O=4
4. **근거**: `O=4 (Transformer Mag: 철심 손실 이상 0.8%, 0.5-2% 구간)`

**D 값 산정**:
1. 검출 방법: 철심 손실 시험 전수
2. 매핑: 비파괴 시험 전수, 검출률 80% → D=2-4
3. 일반적 → D=3
4. **근거**: `D=3 (철심 손실 시험 100% 실시, 검출률 80% 추정)`

### 예시 3: 클램프 피로 파손

**WebSearch 결과**:
- CIGRE: "Clamping structure failure rarely reported (<0.2%)"
- IEEE: "No standard detection method during manufacturing"

**O 값 산정**:
1. 고장률: <0.2%
2. 매핑: <0.5% 구간 → O=1-2
3. CIGRE (최고 신뢰도) → O=2
4. **근거**: `O=2 (CIGRE: 클램프 고장 <0.2%, 매우 희귀)`

**D 값 산정**:
1. 검출 방법: 제조 중 검출 대책 없음
2. 매핑: 검출 대책 없음 → D=9-10
3. 운전 중 발견 가능성 → D=9
4. **근거**: `D=9 (제조 중 검출 대책 없음, 운전 중 진동으로 발견)`

## 주의사항 및 FAQ

### Q1: WebSearch에서 정확한 고장률을 못 찾았을 때?

**A1**: 언어 표현 기반 추정 (매핑 테이블 2 사용)

- "commonly reported" → O=7-8
- "occasionally occurs" → O=5-6
- "rarely seen" → O=3-4

**근거 예시**: `O=6 (IEEE 문헌: 가끔 발생하는 고장 형태)`

### Q2: 여러 출처에서 다른 고장률이 나올 때?

**A2**: 신뢰도 높은 출처 우선 + 보수적 접근

1. CIGRE: 1.5%, IEEE: 3.2% → CIGRE 우선
2. 범위 표시: "1.5-3.2% 범위, 보수적으로 O=5"
3. **근거**: `O=5 (CIGRE: 1.5%, IEEE: 3.2%, 보수적 평가)`

### Q3: 검출률을 정확히 모를 때?

**A3**: 검출 방법 기반 표준 검출률 사용 (매핑 테이블 3)

- 육안 검사 5% 샘플링 → 검출률 30-50% (표준)
- **근거**: `D=6 (육안 샘플링 5%, 검출률 40% 추정)`

### Q4: 출처를 명시해야 하나?

**A4**: 필수! O/D 근거란에 반드시 출처 포함

[X] 나쁜 예: `O=5`
[OK] 좋은 예: `O=5 (CIGRE TB 642: 철심 고장 3.2%, 2-5% 구간)`
