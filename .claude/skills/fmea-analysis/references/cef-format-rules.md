# C/E/F열 상세 설명 형식

> **[BLOCKING]** C/E/F열 모두 `\n(상세설명)` 형식 필수!
> 1줄만 작성 시 Excel 생성 금지!

---

## C열 (고장영향) - 영향 + 상세설명 필수!

**형식**: `영향\n(상세설명)`

> [!!] **같은 고장영향 = 동일한 상세설명! (v14 신규)**
> Leader가 L-1.6에서 확정한 고장영향 문자열을 모든 항목에 그대로 사용!
> 고장형태에 따라 상세설명을 변경하면 엑셀 C열 셀 병합 불가!

| 올바른 예시 | 잘못된 예시 |
|------------|------------|
| `권선 지지력 저하\n(압력 부족으로 인한 고정력 상실)` | `권선 지지력 저하` (1줄만!) |
| `절연 파괴\n(유전체 강도 초과로 인한 지락)` | `절연 파괴` (상세 없음!) |
| `과열 손상\n(냉각 불량으로 인한 온도 상승)` | `과열` (1단어만!) |
| 같은 영향 항목 전부 동일 상세설명 | 형태별로 상세설명 다르게 작성! |

---

## E열 (고장형태) - 태그 + 현상 + 상세 + 판단근거 필수!

**형식**: `태그: 현상\n(상세설명)\n(판단근거)`

| 올바른 예시 | 잘못된 예시 |
|------------|------------|
| `부족: 이완\n(클램프 체결력 부족으로 인한 간극 발생)\n(부족: 고정된 위치가 부족하면 이완 발생)` | `부족: 이완` (1줄만!) |
| `과도: 과체결\n(토크 과다로 인한 변형 발생)\n(과도: 체결력이 과하면 재료 손상)` | `과도: 과체결` (상세 없음!) |
| `유해: 열점발생\n(접촉 불량으로 인한 국부 과열)\n(유해: 정상 기능 수행 시 부작용 발생)` | `유해: 열점` (판단근거 없음!) |

### E열 필수 태그

| 태그 | 질문 | 예시 |
|------|------|------|
| `부족:` | 기능 결과물이 부족하면? | 부족: 적층이완 |
| `과도:` | 기능 결과물이 과하면? | 과도: 과체결 |
| `유해:` | 기능 정상인데 부작용? | 유해: 열점발생 |

---

## F열 (고장원인) - 단계 + 원인 + 상세설명 필수!

**형식**: `단계: 원인\n(상세설명)`

| 올바른 예시 | 잘못된 예시 |
|------------|------------|
| `설계: 수축율 미반영\n(프레스보드 수축율 3-5% 미고려)` | `설계: 수축율 미반영` (1줄만!) |
| `재료: 밀도 부족\n(프레스보드 밀도 1.0g/cm3 미만)` | `재료 불량` (단계 구분 없음!) |
| `제작: 토크 미달\n(체결 토크 85Nm 미준수)` | `제작: 토크 미달` (기준값 없음!) |

### F열 라이프사이클 단계

| 단계 | 비율 | 예시 원인 |
|------|------|----------|
| 설계: | 15-25% | 사양 오류, 계산 오류, 재료 선정 |
| 재료: | 20-30% | 수입검사 미흡, 재질 불량, 규격 오류 |
| 제작: | 25-35% | 작업 미숙, 설비 불량, 공정 이탈 |
| 시험: | 15-25% | 시험 누락, 판정 오류, 환경 조건 |

```
[C/E/F 상세설명 검증]
- [ ] C열: 영향 + \n + (상세설명) 형식
- [ ] E열: 태그 + 현상 + \n + (상세) + \n + (판단근거) 형식
- [ ] F열: 단계 + 원인 + \n + (상세설명) 형식
- [ ] 모든 항목에 \n 포함 (1줄 작성 금지!)
```

## [X] 금지 (BLOCKING)

- E열 판단근거 누락 (예: `부족: 이완\n(설명)` - 3줄째 없음!)
